# Инструкция

## Запуск сборки приложения
### Требование:
* Наличие Linux подобной системы
* Открытие терминала в папке проекта

### Шаги:
1. Ввести команду в терминале
```shell
./BUILD.sh
```

### Шаги:
1. Ввести команду в терминале
