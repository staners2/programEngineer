# Инструкция

## Запуск сборки приложения
### Требование:
* Наличие Linux подобной системы
* Открытие терминала в папке проекта

### Шаги:
1. Ввести команду в терминале
```shell
./BUILD.sh
```

## Запуск тестов приложения
### Требование:
* Наличие Linux подобной системы
* Открытие терминала в папке проекта

### Шаги:
1. Ввести команду в терминале
```shell
./TEST.sh
```

## Запуск самого приложения
### Требование:
* Наличие Linux подобной системы
* Открытие терминала в папке проекта

### Шаги:
1. Ввести команду в терминале
```shell
./RUN.sh {params}
```

2. Список параметров:
```shell
# Логин
-login ... 
# Пароль
-password ...
# Роль
-role ...
# Ресурс
-resourse ...
# Дата начала
-ds ...
# Дата завершения
-de ...
# Объем
-vol ...
```
